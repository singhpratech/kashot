KASHOT Green Icon Pack

This pack uses the green/teal KA mashup icon concept.

Included:
- kashot-icon-master.png: 1024x1024 transparent PNG master.
- Kashot.ico: Windows ICO bundle with 16, 20, 24, 32, 40, 48, 64, 96, 128, 256 px.
- windows_png/: individual Windows PNG sizes.
- macos_iconset/Kashot.iconset/: PNGs ready for macOS iconutil:
  iconutil -c icns Kashot.iconset
- macos_menubar_template/: black alpha template PNGs for macOS menu/status bar.
- linux_hicolor/: freedesktop hicolor directory structure.
- web_pwa_favicon/: favicon and PWA PNG sizes.
- android/: Android launcher and Play Store sizes.
- ios/: common iOS/App Store sizes.

Important:
The source is raster AI artwork. It has been resized/exported for the required platforms.
For absolute best Windows tray readability at 16/20/22/24 px, create a separate simplified vector/silhouette micro-icon later.
